# TDD Workshop
TDD Workshop showing how it works using a hands-on approach

Given a JSON
```
{
    fullName: 'Rene Enriquez',
    role: 'Developer'.
    birthday: '28-01-1987'
}
```

Return another JSON like:
```
{
    firstName: 'Rene',
    lastName: 'Enriquez',
    role: 'Developer'.
    age: 34
}
```
